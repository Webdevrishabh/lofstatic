Best Free Mouse Mover Software For Windows

Move Mouse

Move Mouse is an open source mouse mover software for Windows. 

MouseJiggler

MouseJiggler is an open source mouse mover  software that moves the mouse to and fro at the same  spot. 

XuMouse

XuMouse is a free automatic mouse mover for Windows. It gives you two mouse options: Move Mouse and Click Mouse.
