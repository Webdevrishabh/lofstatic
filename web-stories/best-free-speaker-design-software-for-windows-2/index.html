These are basically speaker box design software or say speaker enclosure design software which let you find optimal calculations for different parameters to correctly design loudspeaker enclosures.

Loudspeaker Design Calculations Toolkit

Loudspeaker Design Calculations Toolkit is a free, portable speaker design software for Windows. It contains a set of tabs which can be used for various calculations and conversions related to loudspeakers.

WinISD

WinISD is a good speaker design software for Windows. It lets you design Closed, Vented, and Bandpass boxes. And, you can save the project in its own file format only.

Box-Port Design

Box-Port Design is another good speaker design software for Windows. It helps you build loudspeaker and subwoofer. Iit evaluates Optimized Sealed Profile and shows the calculations on the interface in real time.

Torres Box Tuning Calculator

Torres Box Tuning Calculator lets you calculate box and port dimensions for audio speakers. The main interface contains all input parameters and output calculations. And the speaker box calculation results can be printed or saved as a TXT file.

Boxnotes

Boxnotes is a free speaker enclosure design software for Windows. It finds out exact box dimensions to design speaker cabinets. You can use its Min button to select a speaker with minimum dimensions.

AJ Vented Designer

AJ Vented Designer is a lightweight, portable speaker design software. Based on speaker parameters, it calculates box results including volume (Vb), box tuning frequency (Fb), cutoff frequency (F3), and length.

Ceiling Speaker Configurator

Ceiling Speaker Configurator is a free software which helps you select proper ceiling speakers. It displays calculations like number of speakers required, distance between speakers, and continuous SPL capability. You can generate and view the report.

Ceiling Speaker Wizard

Ceiling Speaker Wizard is a freeware to basically perform calculations for Martin Audio ceiling speakers. These calculations include optimal values of quantity, position, and performance of speakers needed based on your requirements.

WinSpeakerz

WinSpeakerz is a loudspeaker simulation software. The free version is basically a demo version that comes with a demo speaker project which you can edit and analyze. Additionally, it provides Vent and ‘Crossover and Network’ calculators.

Audio Speaker Design CBVBSPK

Audio Speaker Design CBVBSPK is a portable speaker design software that computes calculations for closed box speaker design and vented (ported) box design. It also displays a Frequency Response graph based on entered parameters.
