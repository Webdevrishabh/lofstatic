13 Best Free Internet Usage Monitor Software For Windows

These free Internet Usage Monitor Software let you monitor data consumption on your computer. View daily, weekly, and monthly statistics, to view and export reports.

NetWorx

NetWorx keeps a record of the total data used and shows the total data transferred along with data usage on hourly, daily, weekly, monthly, and custom basis.

Glasswire

Glasswire visualizes the data usage with a graphical representation where you can get detailed reports for selected time periods.

FreeMeter

FreeMeter is a freeware to monitor internet usage. This internet usage monitoring software displays the internet activity in real time.

NetMeter Evo

NetMeter Evo is another free internet usage monitor for Windows. The graphical interface displays a timeline with internet usage statistics.

tbbMeter

tbbMeter is a freeware to monitor internet usage. It displays the statistics like download speed, upload speed, total downloaded and uploaded data.

BitMeter 2

BitMeter 2 monitors internet usage, displays real time internet usage activity on your screen. The activity is shown on a graphical timeline.

Network Activity Indicator

Network Activity Indicator lets you monitor internet usage on your computer from Windows system tray.

GabNetStats

GabNetStats shows statistics including instantaneous upload and download speed, average upload and download speed, and total data received and sent for the session.

LanLights

LanLights is a free tool to track internet usage. Apart from the total downloaded and uploaded data, this internet usage monitor also displays the real time upload and download speed.
