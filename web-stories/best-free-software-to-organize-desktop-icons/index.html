These software help you to keep your desktop all cleaned up, organized, and visually attractive.

Nimi Places

Nimi Places is a free software to organize desktop icons in various customizable containers, or boxes.

SideSlide

SideSlide is a configurable software that lets you add program icons, files, folders & RSS Feed on desktop.

Stick

Stick is a free software which contains your desktop icons in retractable tab.

FSL Launcher

FSL Launcher organizes dekstop into four different tabs: Applications, Internet, Games, and Utilities.

ToolBox

ToolBox lets you create boxes on your desktop to organize icons. Create, customize, and add program/files, and folders to ToolBox.

Iconoid

Iconoid is multi-purpose tool that can customize icon, reposition, hide, and more.
