Screen  Resolution  Changer Software

HRC - Hotkey Resolution Changer

HRC – Hotkey Resolution Changer is a free and portable screen resolution changer software for Windows.

jRes

jRes is a portable software and you can use it directly from a flash drive without actually installing it on a system.

XP QuickRes

XP QuickRes is another free and portable screen resolution changer software for Windows.

Resolution Changer SX2

This software allows you to easily modify your screen resolution from the system tray area.

Carroll

It is a simple and easy to use Windows software and its main purpose is to help you quickly change the screen resolution.

Resolution Changer

Resolution Changer is a free screen resolution changer app for Windows 10. This app allows you to set screen resolution of your own choice.

Windows 10 Inbuilt Screen Resolution Changer

Windows 10 also provides an inbuilt Screen Resolution Changer to modify the default screen resolution.
