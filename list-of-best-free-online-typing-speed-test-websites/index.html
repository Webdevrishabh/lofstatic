typing.io

Typing.io is another online typing speed test website. Unlike most of the other typing speed test websites, that test typing speed by letting you text normal English
text, this website focuses on typing speed tests based on programming languages. There are a total of 16 programming languages available (like, C#, jQuery, Java, Erlang, Scala, etc.) and you can choose any of them to test your typing speed.
This website makes you type complete code, including the awkward characters that you normally encounter during programming (like, [{~, etc.). So, if you are a programmer, and want to improve your coding speed, this would be your go to website.
Apart from this, the interface of this website rocks as well. When you  choose a language to do typing speed test, it shows the code on the screen, and the cursor is on the first character of the code. And you have to type right on top of the code! So, unlike many websites in which there is a separate textbox in which you have type, that’s not the case here. This is actually pretty advantageous, as you can keep your eyes right where you are typing, and you will also be able to see what you have to type next. And another interesting feature of this website is the way it handles errors. In our normal course of typing, whenever we make a typing mistake, we normally type a few characters, before we realize the mistake. The same is the case with this website. It lets you type a few characters after the mistake, in the hope that you will notice the mistake and fix it yourself, and if you don’t, it stops you there.
Once you are done with the test, it reports regular metrics like WPM, CPM, etc. In addition to that, it reports some unique metrics as well, like, unproductive keystrokes (basically, the keystrokes that you had to type which you eventually deleted).
The website has a paid version as well, and it comes with a feature of Typo Heatmap, which shows you that which areas of the keyboard you make the most mistakes in. All in all, this is a great website for my fellow programmers.
