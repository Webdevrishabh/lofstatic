How to convert MPG to MP4 using FFMPEG:

  • Launch this software in Command Prompt.
  • After that, enter this MPG to MP4 conversion command: ffmpeg.exe -i Video.mpg OutVideo.mp4
  • Now, just press the Enter button to start the conversion process.

In the above command, Video.mpg is the name of the input file and OutVideo.mp4 is the name of the output file.
